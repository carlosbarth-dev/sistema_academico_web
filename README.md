# Sistema Academico Web

Projeto simples de Sistema Academico Web feito com HTML, CSS e JavaScript.

## Funcionalidades

- Cadastro de alunos
- Cadastro de turmas
- Lancamento de notas
- Consulta de boletim
- Media geral, aprovados e totais do sistema
- Busca e remocao de alunos
- Dados salvos no navegador com localStorage

## Como usar

Abra o arquivo `index.html` no navegador.

## Estrutura

- `index.html`: estrutura da pagina
- `styles.css`: estilos visuais
- `app.js`: regras e armazenamento local
